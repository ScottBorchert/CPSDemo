Welcome to my Angular demo!

Prior to 5/10/2023 I had never written a single line of Angular, nor have I ever seen any Angular code. I created this Angular site based off of the previous years of Full Stack development work I've done in other .NET MVC sites and React Sites.

Here I have:
  • Created a simple chat box. After entering your message it sends the message to a backend Node.js api that will response with some Lorem Ipsum text with your message appended to the end.
  • Published site to a new Azura static website.
  • Added a Github Actions CI/CD to have it publish on commit to the "main" branch.
  • Added a login store that validates credentials with backend Node.js api server.
  • Added a simple view to interact with a webcam.
